About Image Caption Generator

Image caption generator is a process of recognizing the context of an image and annotating it with relevant captions using deep learning, and computer vision. It includes the labeling of an image with English keywords with the help of datasets provided during model training. ResNet is used to train the CNN model and is responsible for feature extraction. The extracted features will be fed to the LSTM model which in turn generates the image caption.
